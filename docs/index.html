سرعت, قدرت و بهینگی در یک کتابخانه ⚡

معرفی

این کتابخانه که از کد های سینک و ایسینک(sync/async) پشتیبانی میکند و با سرعت بالایی میتوانید درخواست ها را ارسال و پردازش کند همچنین با حجم کم آن و وابستگی های کم آن و مصرف پایین رم و پردازنده و ... می تواند برای ربات های سنگین روی سرور های سبک با سرعت ویژه ای شروع به کار کند این کتابخانه مجهز به سیستم لاگ گیری میباشد این کتابخانه به طور اصلی برای بات ها نوشته شده و همچنین از یوزر بات ها هم پشتیبانی می کند. این کتابخانه در حال حاظر در پایتون است و بزودی در سی شارپ هم به مرحله انتشار میرسد

مراحل استفاده

  • با استفاده از Bot Father به آدرس @BotFather در روبیکا یک بات بسازید.
  • توکن دریافتی را کپی کنید و در باقی مراحل از آن استفاده کنید.
  • نحوه نصب کتابخانه در ترمینال :
    • pip install --upgrade fastrub
  • نحوه نصب کتابخانه در ترمینال در با میرور در صورت ملی شدن اینترنت :
    • pip install -i https://mirror-pypi.runflare.com/simple fastrub

شروع سریع 🚀

from fast_rub import Client # ایمپورت کلاس کلاینت از کتابخانه
from fast_rub.types import Message # ایمپورت آپدیت برای تحلیل بهتر پیام ها
import asyncio

async def robot():
  bot = Client("my_bot", run_start=False) # ساخت کلاینت

  await bot.start()

  @bot.on_message()
  async def test_update(msg: Message):
      print(f"msg : {msg}")
      print(f"text msg : {msg.text}")
      await msg.reply("*this* ||is|| a ~~test~~ `from` **fast_rub** !")
  
  bot.run()

asyncio.run(robot()) # اجرا ربات
